«Работа с файлами»
1. «Проверка перед копированием»
Ваша программа должна скопировать файл «отчёт.txt» из папки D:
\Документы в папку D:\Архив.
Задача: Перед копированием проверьте, существует ли исходный файл.
Если его нет, выведите в консоль сообщение об ошибке и завершите
выполнение. Если исходный файл существует, а целевая папка отсутствует
– создайте её с помощью новый Файл("D:\\Архив").СоздатьКаталог().

2. «Копирование всех текстовых файлов»
В папке D:\Документы лежат файлы с разными расширениями (txt, log, csv,
docx, pdf).
Задача: Найдите все файлы с расширениями .txt, .log и .csv и скопируйте их
в папку D:\Тексты. Расширения должны проверяться без учёта регистра
(например, .TXT тоже подходит). Используйте метод Файлы.Найти с
маской *.*, затем в цикле проверяйте расширение каждого файла с
помощью метода Строка.ЗаканчиваетсяНа (с учётом регистра или без). Либо
выполните три отдельных поиска с масками *.txt, *.log, *.csv и скопируйте
найденные файлы. Выведите количество скопированных файлов.

3. «Уборка временных файлов»
В папке D:\Temp накопилось много временных файлов с расширением .tmp.
Задача: Найдите все такие файлы и удалите их. Если файла нет, удаление
должно происходить без выброса исключения (используйте второй
параметр метода Удалить). После удаления выведите в консоль количество
удалённых файлов.

4. «Лог работы программы»
В начале выполнения скрипта создайте временный текстовый файл с
префиксом MyLog_ и
расширением .log (используйте Файлы.СоздатьВременныйФайл).
Задача: Запишите в этот файл три строки:

1. Текущие дату и время
2. Имя пользователя (можно
взять Файлы.ПолучитьДомашнийКаталогПользователя(), но лучше
использовать системное свойство или просто написать фиксированное
имя)
3.Сообщение «Скрипт завершён успешно»

После этого прочитайте файл целиком и выведите его содержимое в
консоль. Убедитесь, что файл автоматически удалится после работы
скрипта (установите параметр автоматического удаления).

5. «Сигнальный файл – мониторинг»
Создайте программу, которая отслеживает появление файла start.txt в
папке D:\Обмен.
Задача: Используйте МониторФайловойСистемы и бесконечный цикл пока
Истина. Когда файл появляется, выведите сообщение «Обнаружен сигнал
начала обработки» и удалите этот файл. Затем создайте файл finished.txt с
текстом «Обработка завершена». Через 5 секунд (можно
использовать Ожидать(5с)) остановите мониторинг и выйдите из
программы.

6. «Чтение оценок из CSV»
В файле D:\Оценки.csv (кодировка UTF-8) хранятся данные в формате:
Фамилия;Имя;Математика;Русский;Физика
Пример:
Иванов;Иван;5;4;5
Петрова;Анна;4;5;4
Задача: Прочитайте файл построчно, разделите каждую строку на части с
помощью Разделить(";") и выведите в консоль таблицу в виде:
«Иванов Иван: математика – 5, русский – 4, физика – 5»
Для каждой строки.

7. «Запись результатов теста»
После выполнения теста у вас есть переменные: ИмяУченика =
"Сидоров", Баллы = 85, Максимум = 100.
Задача: Запишите в файл D:\Результаты.txt (в кодировке ANSI, "ansi-1251")
строку формата:
[текущая дата и время] Сидоров: 85/100 баллов.
Файл должен дополняться (если файл уже существует, новая строка
добавляется в конец).
Подсказка: используйте ПотокЗаписи с параметром добавить = Истина при
открытии или откройте поток на запись с флагом добавления.

8. «Копирование только новых файлов»
В папке D:\Новые лежат файлы, которые нужно скопировать в D:\Бэкап. При
этом копировать следует только те файлы, которых ещё нет в целевой
папке (пропускать существующие).
Задача: Настройте НастройкиКопированияФайлов с ПропускатьСуществующие(Ис
тина) и скопируйте все файлы из D:\Новые в D:\Бэкап. Выведите сообщение
«Скопировано X файлов» (X – количество реально скопированных, без
учёта пропущенных).
Подсказка: метод Файлы.Скопировать с настройками не возвращает
количество, поэтому придётся сначала найти список файлов, а затем
копировать по одному, отслеживая успех через обработку исключений или
предварительную проверку существования.

9. «Перемещение старых логов»
В папке D:\Логи хранятся файлы с расширением .log. Нужно переместить все
логи, которые были изменены более 7 дней назад, в папку D:\АрхивЛогов.
Задача: Для каждого файла получите дату последнего изменения
(Файл.ПоследнееИзменение), сравните с текущей датой. Если разница больше
7 дней – переместите файл. Создайте архивную папку, если её нет.

10. «Резервное копирование с датой»
Создайте копию файла D:\Важное\проект.bsl в той же папке, добавив к имени
текущую дату в формате ГГГГ-ММ-ДД. Например, проект_2025-05-20.bsl.
Задача: Проверьте, существует ли исходный файл. Если да, то сформируйте
новое имя, используя ДатаВремя.Сейчас() и метод Представление("гггг-ММ-дд").
Скопируйте файл. Также создайте текстовый журнал backup.log, в который
запишите строку: «[дата время] Создана копия проекта.bsl → новое_имя».
Если исходного файла нет – запишите в журнал ошибку.
